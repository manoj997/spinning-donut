# Clone this project

```bash
git clone https://github.com/manoj997/spinning-donut
```

# Change directory to this project

```bash
cd spinning-donut
```

# Compile the file

```bash
gcc -o donut.c donut -lm
#ignore the warnings

```

# Run the file

```bash
./donut

```
